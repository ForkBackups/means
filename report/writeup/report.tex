\begin{abstract}
We present \emph{Moment Expansion Approximation for iNference and Simulation} (\means) -- a python package for stochastic modelling based on \acrfull{mea},
\acrfull{lna} and \acrfull{ssa}. 
We aimed to provide a performant and user-friendly tool that integrates well with IPython interactive environment.
We implemented novel developments to \gls{mea}, such as parametric moment closure methods.
In addition, our package provides a straightforward way to apply these approximations for parameter inference.
We took great care of documenting this package clearly as well as provided a complete user tutorial.
We also included support for \gls{sbml} standard for studying available biological systems.
We later used this package to conduct further analysis of \gls{mea}, and
found that using a higher maximal moment order did not necessarily improve accuracy.
Moreover, we discovered that \acrlong{mea} was better in certain regions of the parameter space than others.
Some parameter sets also resulted in systematic failure of the numerical solvers.
These limitations may provide direction for improvements of simulation and inference using \gls{mea}.
We believe \means{} will a valuable package for researchers to studying non-linear kinetic systems.
\end{abstract}
